Program Name: Vehicles\
\
This program acts as a system of vehicles where the user will be shown a menu of options to view various types of vehicles contained in the system. \
\
To run the program:\
Step 1: Change directories to the directory above the directories edu and classes, followed by the enter button.\
Ex. Type "cd /home/temps/ottofy/cs1302/Project1", followed by the enter button.\
\
Step 2: Compile the program by typing "javac -d classes " followed by the address to the directory called vehicles, followed by the enter button.\
Ex. Type "javac -d classes /home/temps/ottofy/cs1302/Project1/edu/uga/cs1302/vehicles/*.java" followed by the enter button.\
\
Step 3: Run the program by typing "java -cp classes Vehicles", followed by the enter button.\
Ex. Type "java -cp classes Vehicles", followed by the enter button.\
\
The program will prompt you to:\
Press 1 to see how many vehicles are in the system.\
Press 2 to see the name and kind of each vehicle.\
Press 3 to see which vehicles can fly.\
Press 4 to see which vehicles can float.\
Press 5 to see which vehicles can float AND fly.\
Press 6 to see a description of each vehicle.\
Press h to see a brief help information for your system.\
Press q to terminate the program.\
\
Note: The enter button must be hit after each selection to activate the action.}
